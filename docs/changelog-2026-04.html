Changelog

April 2026

🐍

Python SDK

Added Python SDK for humanenv with skill integration for AI agents. Enables Python applications to securely fetch secrets using the same security model as the Node.js SDK.

�

Local Serverless Mode

Implemented serverless mode with SQLite storage for single-machine setups. Includes CLI commands for init, get, set, projects, envs, apikeys, whitelist, export, and import with mnemonic-based authentication.

�

Enhanced API Key Management

Added optional API key naming, last-used tracking, and project-level requireApiKey setting. Improved key usage tracking with periodic flush and shutdown cleanup.

�

Security Enhancements

Added fingerprint verification toggle with whitelist tab visibility control. Implemented temporal PK persistence with project-based salt and global settings key-value store.

🧪

Comprehensive Testing

Added 354 tests across client, server, and CLI packages. Includes unit tests for MongoProvider, SqliteProvider, HumanEnvClient lifecycle, route handlers, and WebSocket connections.
